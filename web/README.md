# MiniK8s Web Dashboard

This is the web frontend for the MiniK8s platform. It provides a real-time dashboard to monitor and manage the cluster, nodes, and pods.

## Features
- Node and pod overview
- Real-time metrics and charts (CPU/RAM)
- Logs streaming
- Create and manage pods
- Cluster health monitoring

## Technologies
- React
- Vite
- TypeScript
- (Optionally) TanStack Query, Charts.js, WebSockets/SSE

## Getting Started

```bash
npm install
npm run dev
```

The app assumes the backend API is available on `http://localhost:8080` by default.
